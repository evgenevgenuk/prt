# ◈ PRT MENU — Portal 2 Cheat Overlay

> Графічний оверлей для Portal 2 який висить поверх гри і дозволяє вводити команди в консоль одним кліком.

---

## 🚀 Встановлення

### Вимоги
- **Python 3.8+** — [python.org](https://python.org)
- **Windows** (тільки Windows, використовує win32api)
- **Portal 2** у **Windowed** або **Borderless** режимі

### Запуск
```bash
pip install pywin32 keyboard
python prt_overlay.py
```

> Бібліотеки встановлюються автоматично при першому запуску.

---

## 🎮 Як користуватись

1. Запусти Portal 2 у **віконному режимі** (Options → Video → Windowed)
2. Запусти `prt_overlay.py`
3. У полі **ПРОЦЕС** вибери `Portal 2` зі списку (знаходиться автоматично)
4. Натискай кнопки — скрипт сам відкриє консоль, введе команду і закриє

| Клавіша | Дія |
|---------|-----|
| `Insert` | Показати / сховати меню |
| Drag за заголовок | Перемістити меню |
| `✕` | Закрити програму |

Toggle-кнопки які вмикаються/вимикаються кліком:

| Кнопка | Команда | Опис |

| SV_CHEATS | `sv_cheats 1/0` | Вмикає доступ до читів |

| NOTARGET | `notarget` | Турелі тебе не бачать |

> ⚠️ При першому вмиканні будь-якого читу з'являється попередження про `sv_cheats`

### BIND — Прив'язка клавіш

1. Введи клавішу (наприклад `f5`, `k`, `mouse3`)
2. Введи команду або вибери з швидкого вибору
3. Натисни **ЗРОБИТИ BIND**

**Автоматична конвертація української розкладки:**
```
д → l,  п → g,  н → y,  і → s ...
```
Якщо введеш `д` — скрипт автоматично замінить на `l` і напише `bind l <команда>`

## ⚙️ Технічні деталі

```
prt_overlay.py використовує:
├── tkinter        — графічний інтерфейс
├── win32gui       — знаходить вікно Portal 2
├── win32api       — симулює натискання клавіш (VK коди)
├── keyboard       — глобальний hotkey Insert
└── threading      — відправка команд без заморозки UI
```

**Як відправляється команда:**
1. Меню ховається
2. `SetForegroundWindow` — фокус на Portal 2
3. `keybd_event(0xC0)` — відкриває консоль (~)
4. `keyboard.write(cmd)` — вводить текст
5. `VK_RETURN` — Enter
6. `keybd_event(0xC0)` — закриває консоль
7. Меню повертається

---

## ⚠️ Важливо

- Portal 2 має бути у **Windowed** режимі — у Fullscreen скрипт не може надсилати команди
- `sv_cheats 1` потрібен для більшості команд — скрипт запитає дозвіл при першому використанні
- Деякі команди спавну (`ent_create`) не працюють в **challenge mode** без sv_cheats
- Гелі (кульки) з'являються але не фарбують підлогу на картах де поверхні не мають тегу `paintable`
